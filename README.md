Digital Twin Sustainability Simulator (C++)

Overview

The Digital Twin Sustainability Simulator is a menu-driven C++ application that simulates and monitors sustainability metrics such as energy consumption, water usage, and waste generation. It helps users analyze resource utilization and evaluate overall sustainability performance.

Features

- Record monthly energy, water, and waste data
- View a detailed sustainability report
- Calculate an overall sustainability score
- Display performance status (Excellent, Good, Needs Improvement, or Poor)
- Interactive and easy-to-use menu-driven interface

Technologies Used

- C++
- Structures ("struct")
- Vectors
- Loops and Conditional Statements
- Basic Data Analysis

How to Run

1. Compile the program using any C++ compiler.
2. Run the executable.
3. Enter monthly sustainability data through the menu.
4. View reports and check the sustainability score.

Future Improvements

- File handling for persistent data storage
- Graphical data visualization
- Carbon footprint estimation
- Monthly trend analysis and predictions
